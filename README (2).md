# LapKr · Presiones

Captura rápida de presiones de neumáticos pre-grid.

## Estado actual

- **Local-only** — los datos viven en `localStorage` del navegador
- Sin login, sin sync entre dispositivos
- Mobile-first, optimizado para uso con guantes en pre-grid

## Pendiente · Ronda 2

- Migración a Firebase (sync multi-device)
- Login + permisos del equipo LapKr
- Pilotos y compuestos editables (con aprobación admin)
- 8 sesiones de vida de llanta
- Integración con dashboard

## Acceso

Desde el dashboard de LapKr o directo: `lapkr-presiones.vercel.app`
